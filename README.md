# android-lecture
android lecture notes
powered by [Remark](https://github.com/gnab/remark)

안드로이드 강의 자료

## How to make and view lecture notes
```
$ git clone https://github.com/jyheo/android-lecture.git
$ cd lecture-notes
$ make
```
* Open .html with your web browser.
